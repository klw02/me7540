# heat2d

Simple finite element heat equation solver

## Installation

## One time setup

```console
git clone git@github.com:tjfulle/me7540
cd me7540
python3 -m venv venv
source venv/bin/activate
cd Exercises/heat2d
python3 -m pip install -e .
```

## Run an example

```console
python3 -m heat2d verify
```
